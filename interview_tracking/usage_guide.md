# 使用方式

## 你以后怎么和我配合

你每次可以直接发以下任意一种内容给我：

1. 一整段面试对话
2. 你回忆出来的题目和自己的回答
3. 某一题你觉得答得不好的片段
4. 最新面试进度更新
5. 某家公司下一轮前的准备需求

## 我默认会帮你做的事

1. 提炼这场面试的核心内容
2. 分析面试官真实在考什么
3. 找出你回答里的薄弱点
4. 给出更适合面试场景的优化版回答
5. 总结下一轮最该准备的内容
6. 必要时帮你同步成追踪记录

## 推荐提问方式

你可以直接这样发：

`这是我今天一面的对话内容，帮我复盘并优化回答。`

`这是 XX 公司嵌入式岗位二面的问题和我的回答，帮我总结面试官关注点。`

`这是我最近所有面试进度，帮我汇总短板并给出下周计划。`

`帮我把这场面试整理成可持续跟踪的复盘记录。`

## 文件建议

- 总表用：`master_tracker.md`
- 单场复盘：复制 `interview_log_template.md`
- 如果后面公司变多，可以按公司单独建文件
