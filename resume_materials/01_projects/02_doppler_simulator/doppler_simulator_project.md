# 24GHz FMCW 毫米波多普勒模拟与距离-多普勒热图验证系统

## 项目简介

基于 `24GHz FMCW` 雷达平台与 Python 模拟器，设计并实现距离-多普勒热图验证系统，完成板端数据采集、二维 FFT 处理、虚拟通道补偿、串口传输、上位机可视化与多普勒信号仿真验证闭环。

## 工作描述

负责嵌入式端开发，并配套完成上位机联调工具。基于 `AT32F435 / RKB2301` 平台完成板级初始化、AFE/ADC 采样链路搭建和 frame-chirp 时序调度；围绕 `2Tx3Rx TDM-MIMO` 数据流实现直流去除、Hamming 窗、Range FFT、6 个虚拟通道相位补偿、静态杂波抑制与 Doppler FFT，生成 `Range-Doppler Map`；设计自定义 `TLV` 协议，通过 `UART5@921600` 输出原始采样和热图数据，支撑板端调试与链路联调；配套开发 Python 上位机和仿真脚本，用于串口解包、热图显示以及距离/速度映射验证。

## 简历版

### 标准版

**24GHz FMCW 毫米波多普勒模拟与距离-多普勒热图验证系统**

- 基于 `AT32F435 / RKB2301` 完成 `24GHz FMCW` 雷达嵌入式端开发，负责 BSP 初始化、AFE 启停、ADC 采样链路、frame/chirp 定时调度及中断消息处理。
- 面向 `2Tx3Rx TDM-MIMO` 数据链路实现 DC 去除、Hamming 窗、Range FFT、6 个虚拟通道固定相位补偿、静态杂波抑制与 Doppler FFT，生成 `Range-Doppler Map`。
- 设计 `RawData / RangeDopplerMap` 双模式 `TLV` 协议，通过 `UART5@921600` 输出原始采样和二维热图，支撑板端调试、数据回传与上位机联调。
- 配套开发 Python 上位机和仿真脚本，用于串口帧同步、热图可视化及距离频率/多普勒频率映射验证，辅助板端算法调试。

### 精简版

**24GHz FMCW 毫米波多普勒模拟与距离-多普勒热图验证系统**

- 基于 `AT32F435` 完成 `24GHz FMCW` 雷达嵌入式处理链路开发，实现 `2Tx3Rx TDM-MIMO` 数据采集、Range FFT、相位补偿、静态杂波抑制与 Doppler FFT。
- 设计 `TLV` 数据回传方案，通过 `UART5` 输出 `RawData` 和 `Range-Doppler Map`，配套上位机完成帧同步、解包和热图显示。
- 使用 Python 仿真脚本验证 IF 频率、目标距离、目标速度与热图峰值位置之间的映射关系，辅助板端链路调试。

### 差异化写法

#### 1. 雷达验证平台版

**24GHz FMCW 毫米波多普勒模拟与距离-多普勒热图验证系统**

- 面向 `24GHz FMCW` 雷达距离-速度测量场景，搭建板端处理、串口回传、上位机显示与仿真分析一体化验证平台。
- 围绕 `2Tx3Rx TDM-MIMO` 数据流完成虚拟通道组织、固定相位补偿与二维热图生成，验证多普勒信息在距离-速度平面上的成像效果。
- 建立目标距离、目标速度、IF 频率与多普勒频率之间的映射关系，辅助分析热图峰值位置与参数配置的对应关系。

#### 2. 建模仿真版

**24GHz FMCW 毫米波多普勒模拟与距离-多普勒热图验证系统**

- 基于 `Python` 编写多普勒与 `Range-Doppler` 仿真脚本，模拟距离频率和多普勒频率叠加过程，验证二维 FFT 后热图峰值分布。
- 结合板端 `24GHz FMCW` 参数配置，对采样率、chirp 数、带宽、距离分辨率和速度分辨率进行联动分析，支撑参数选型与结果解释。
- 将仿真结果与板端输出热图对照，形成“原理分析 - 仿真验证 - 实际采集”闭环，提高算法验证效率。

#### 3. 系统联调版

**24GHz FMCW 毫米波多普勒模拟与距离-多普勒热图验证系统**

- 设计 `RawData / RangeDopplerMap` 双模式数据链路和 `TLV` 协议，完成板端原始采样、二维热图与上位机可视化之间的稳定联调。
- 开发串口控制与显示工具，支持工作模式切换、帧同步、数据解包和热图展示，提升雷达中间结果调试效率。
- 配套搭建多普勒注入与参数控制工具，用于验证频率叠加逻辑和系统状态切换过程。
