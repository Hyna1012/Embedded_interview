# 基于毫米波雷达与轻量化神经网络的跌倒检测系统

## 项目简介

基于毫米波雷达与轻量化神经网络，设计并实现跌倒检测系统，完成雷达数据采集、信号处理、模型训练、嵌入式端部署及上位机联调，构建从算法验证到端侧落地的完整闭环。

## 工作描述

负责嵌入式端开发与算法部署。基于 `AT32F435 / RKB2301` 平台完成板级初始化、雷达数据采集链路搭建及底层驱动调试；基于 `CMSIS-DSP` 实现 ADC 数据预处理、窗函数、FFT、静态杂波抑制/MTI 及距离向量提取等雷达信号处理流程；完成 `CNN / GRU / LSTM` 等轻量化模型的训练、参数导出与嵌入式端推理部署，结合存在性检测与时序滑窗实现跌倒分类；设计自定义 `TLV` 串口通信协议，输出 IF 信号、MTI、距离向量及分类概率；使用 `Python + PyTorch + PySide6` 开发数据处理、模型验证与上位机可视化工具，支持串口解包、结果展示和联调分析。
